При запуске приложения появляется главное меню.
Список доступных команд можно посмотреть путем ввода команды "help".
Каждое действие сопровождается пояснением.
В большинстве случаев сначала вводится команда, а потом необходимые данные.

Пример создания пользователя:
1. Ввести команду cr
2. Ввести имя
3. Ввести пароль

Во время игры в крестики нолики координаты указываются через пробел, иначе программа
не примет данные и попросит повторить ввод. Пример ввода координат: A 0;
